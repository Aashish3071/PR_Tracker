Report Template Guide

This guide explains how to use custom templates to automatically generate client reports.
Overview

The PR Tracker's automated report generation system allows you to upload your own Excel templates (like Sony's) and have the system automatically fill them with data from your coverage entries. This saves hours of manual work and ensures consistency across all your client reporting.

Step 1: Upload Your Template

Upload your client-specific Excel template with placeholders for data.

  1. Navigate to Report Templates in the main navigation
  2. Click Add New Template
  3. Upload your Excel file
  4. Name your template (e.g., "Sony Template")
Step 2: Configure Mapping

Map the placeholders in your template to data fields in PR Tracker.

  1. After uploading, you'll see the mapping interface
  2. Match each placeholder with the corresponding data field
  3. For example, map {{CLIENT_NAME}} to the client record
  4. Save your mapping configuration
Step 3: Generate Report

Generate your report with a single click.

  1. Go to Generate Reports in the navigation
  2. Select your template from the dropdown
  3. Set the date range for the report period
  4. Click Generate Report
Step 4: Download & Share

Review, download, and share your completed report.

  1. Preview the generated report
  2. Make any final adjustments if needed
  3. Click Download Report
  4. Share the Excel file with your client
Template Requirements
Placeholder Type Example Description
Text Placeholders {{CLIENT_NAME}} Simple text replacements for client names, dates, etc.
Table Placeholders {{COVERAGE_TABLE}} Entire tables of data, like coverage listings
Numeric Placeholders {{TOTAL_AVE}} Numbers and calculations like total AVE, impressions, etc.
Date Placeholders {{REPORT_DATE}} Date values that will be formatted according to your template
Tips for Success
Use Consistent Placeholders

Use the same format for all placeholders, like {{PLACEHOLDER_NAME}}

Preserve Formulas

Your Excel formulas will remain intact, only data cells will be updated

Test with Small Date Ranges

Run a test report with a small date range to verify everything maps correctly

Keep Formatting

All your template's formatting, charts, and styling will be preserved
